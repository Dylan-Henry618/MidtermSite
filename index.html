What's up with all these REmakes?

We live in an age where it's hard to make original content, so we go to remake some pretty good games like Crash Bandicoot.

Some of these remakes are for games that everyone usually really wants but never expected it. If done properly, these remasters can be even better than the original. Whether it's adding in new content that doesn't break the flow of the game or in some extreme cases, taking out or redoing content that the fan base didn't like.


And then there are games that are probably better left untouched...


So what goes for a good or bad remaster? Or really what games in recent history are worth talking about at the moment that might explain why companies really want to keep making remakes? Let's go through and find out!!!
